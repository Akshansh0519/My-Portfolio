## ✨ Portfolio Highlights & Features

This portfolio is built from the ground up using **React 18**, **Vite**, and **Framer Motion**, designed around modern **Swiss / Brutalist typography** and **glassmorphism** aesthetics.

* 🎬 **Interactive AI Video Intro**: Features a custom-trained AI talking avatar intro synchronized with dual career pathways (**SDE** vs **AI/ML** downloadable resumes).
* ⚡ **Layered Typography & Parallax Depth**: Implements award-winning depth-clipping typography and dynamic scroll-driven parallax effects.
* 🧠 **AI & Research Showcase**: Dedicated showcase of academic research, including my **IEEE Paper** on AI-driven diagnostics, **ML Internships**, and PyTorch hackathon victories.
* 🛠️ **Comprehensive Project Hub**: Deep-dive interactive modals and source code repositories for major engineering systems:
  * 🃏 **BluffBuster**: Multi-agent AI poker bot powered by LangGraph, Groq Llama-3, and PyTorch (Meta Hackathon Winner).
  * 🛍️ **ShopSense**: Intelligent AI shopping assistant utilizing RAG (Retrieval-Augmented Generation) and vector search.
  * 🔄 **SyncNexus**: Real-time collaborative IDE and workspace powered by WebSockets, Operational Transformation, and Node.js.
  * 🔗 **LinkForge**: High-concurrency custom URL shortener with real-time click analytics and Redis caching.
  * 🌊 **OrqFlow**: Distributed workflow automation and job orchestration engine.
* 📬 **Seamless Communication**: Fully integrated **EmailJS** messaging system with intelligent fallback mechanisms and direct links to competitive programming profiles.

## 🛠️ Tech Stack & Architecture

* **Animations**: Framer Motion (Parallax, Scroll Transforms), Animate On Scroll (AOS)
* **Assets**: Custom Lucide SVG icons, multi-format media optimization

### **AI & Backend Technologies (Featured Projects)**
* **AI & ML**: Python, LangGraph, PyTorch, RAG, Groq Llama-3, Vector Databases (Pinecone/Chroma)
* **Backend**: Node.js, Express.js, WebSockets, REST APIs
* **Databases & Cache**: PostgreSQL, MongoDB, Redis
* **DevOps & Cloud**: Docker, Git, Linux, Vercel / Netlify Edge CDN

## 🚀 Getting Started (Local Development)

To run this project locally on your machine:

### 1. Clone the repository
```bash
git clone https://github.com/Akshansh0519/My-Portfolio.git
cd My-Portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.

### 4. Build for production
```bash
npm run build
```
